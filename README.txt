Решение тестового задания в компанию Neurus. Задание сделано не идеально и возможно не правильно, но оно будет дорабатываться.

Первый пункт состоит из 2-ух папок:
1) clear - здесь собранный код, приведённый к удобному чтению;
2) rough - черновой вариант, где прикреплены комментарии, в большей степени сделано для себя.

Второй пункт состоит из 2-ух папок:
1) transfer_learning - здесь, я постарался разобраться с Tensorflow Object Detection Api, подготовил все файлы, поставил модель (удалена из репозитория в целях экономия места), но до конца не разобрался с файлом конфигурации и как запускать обучение;
2) VGG16 - думал реализовать задание через обученную модель VGG16, но перешёл на Tensorflow Object Detection Api.

Идея запуска проекта из папки transfer_learning:
Переходим в папку Scripts и запускаем:
1) pation_dataset.py - для сортировки изображений по папкам (train, test);
2) generate_tfrecord.py - два раза, для генерации файлов нужного формата (1 раз для папки train, 2 для папки test);
3) model_main_tf2.py;
4) exporter_main_v2.py.
